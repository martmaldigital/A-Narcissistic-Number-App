A Narcissistic Number App

Andela Interview Test

A Narcissistic Number is a number which is the sum of its own digits, each raised to the power of the number of digits in a given base.
This app takes a positive integer(base 10) and checks if it is a narcissistic number or not.
For example: 153 (3 digits): 1^3 + 5^3 + 3^3 = 1 + 125 + 27 = 153
1634 (4 digits): 1^4 + 6^4 + 3^4 + 4^4 = 1 + 1296 + 81 + 256 = 1634
